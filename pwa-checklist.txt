🚀 PASOS DE INSTALACIÓN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Instalar dependencias:
   > npm install

2. Generar iconos:
   > npm run generate:icons
   
   O manualmente:
   - Abre: http://tu-dominio/images/generate-icons.html
   - Descarga todos los iconos
   - Guárdalos en: public/images/

3. Compilar assets:
   > npm run build

4. Verificar en DevTools:
   - Chrome DevTools > Application
   - Verificar Manifest
   - Verificar Service Worker
   - Verificar Icons

5. ¡Instalar la app!

🆘 TROUBLESHOOTING COMÚN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problema: Service Worker no se registra
Solución: 
  - Verifica HTTPS (o localhost)
  - Revisa consola para errores
  - Limpia caché del navegador

Problema: No aparece prompt de instalación
Solución:
  - Verifica manifest.json accesible
  - Verifica que existan todos los iconos
  - Espera algunos segundos/interacciones

Problema: Cambios no se reflejan
Solución:
  - Incrementa versión en sw.js
  - Desregistra SW en DevTools
  - Hard refresh (Ctrl+Shift+R)

Problema: Iconos no aparecen
Solución:
  - Verifica rutas en manifest.json
  - Genera iconos con npm run generate:icons
  - Verifica permisos de carpeta
